## Typora+Github+jsDeliver搭建图床 

### 一. 简介

#### 1. 什么是图床?

图床一般是指储存图片的服务器,有国内和国外之分.国外的图床由于有空间距离等因素决定访问速度很慢影响图片显示速度.国内也分为单线空间、多线空间和cdn加速三种.

> 自定义域名:https://cdn.jsdelivr.net/gh/chanwanxiang/imageHosting
